# 中文 Code Review 產生器

## 使用方式
在 Cursor Agent 輸入：
- 「幫我 review 這段程式碼」
- 「幫我 review src/webhook.ts」

## 輸出格式
## ✅ 優點
- <值得肯定的地方>

## ⚠️ 建議改善
- <可以更好但非必要的項目>

## ❌ 需要修正
- <有問題、有風險、必須改的項目>

## 總結
<整體評估一句話>
