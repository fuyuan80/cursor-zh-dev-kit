# 中文 PR 說明產生器

## 使用方式
在 Cursor Agent 輸入：
- 「幫我寫 PR 說明」
- 「幫我寫 PR 說明，branch 是 fix/payment-callback，base 是 main」

## 輸出格式
## 變更摘要
<一句話描述這個 PR 的目的>

## 主要改動
- <改動項目 1>
- <改動項目 2>

## 測試方式
- <如何驗證這個改動>

## 注意事項
- <部署注意、相依套件、breaking change 等>
